Create a Burger app using express-handlebars, so that you can pick a burger, add a burger and also devour it.

## Instructions
 Pick a Burger and Devour it.

Techologies:
    - JavaScript
    - Node JS
    - VS-Code Editor
    - MySQL
    - MySQL WorkBanch
    - GitHUb Repository
    - npm packages


Locally you can run it on the node.js console node server.js
